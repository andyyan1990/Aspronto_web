== Description ==
Corporate Landing Page is a professional WordPress theme suitable for corporates, small and medium business as well as creative agencies. This theme is simple, attractive and responsive and comes with a custom homepage template with lots of customization options like About section, wide screeen slider, contact section, etc. Besiedes that, theme has footer widgets, full width page template and custom area to showcase your contact info in header. Corporate Landing Page is also compatible with WooCommerce so you now easily sell your products online. Theme demo is available at: http://demo.themethread.com/corporatelandingpage/

Theme demo: http://demo.themethread.com/corporatelandingpage/

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
